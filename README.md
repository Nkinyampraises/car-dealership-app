# xrwvm-fullstack_developer_capstone

## Project Name: Best Cars Dealership

## Project Description
Best Cars Dealership is a full-stack web application built with Django and React. It allows users to browse car dealerships across the United States, read and post reviews, and analyze review sentiments using IBM Watson NLU. The application features user authentication, dealer search by state, and a complete review management system.

## Technologies Used
- **Frontend:** React.js, HTML, CSS, Bootstrap
- **Backend:** Django, Python
- **Database:** SQLite / PostgreSQL
- **Sentiment Analysis:** IBM Watson NLU
- **Deployment:** IBM Cloud Code Engine
- **CI/CD:** GitHub Actions

## Features
- Browse all car dealerships nationwide
- Filter dealers by state
- User registration and login
- Post and view dealer reviews
- Sentiment analysis on reviews
- Admin panel for managing data
